Pokemon text game

The aim is to navigate as Ash (A) towards Pikachu (P), whilst avoiding the enemies Zubat (Z) and Doduo (D). 

Upon prompting of maze file please type the full path: eg.  C:\Other\csse1001\Assignments\POKEMON2_25_08_12_43\maze1.txt
